# カラーパレット

## デザインコンセプト

料理アプリらしい温かみを持ちつつ、ミニマルで洗練された印象を両立する。
ウォームトーンを基調とし、自然で食欲を刺激するカラーリングを採用する。

## カラー定義

### プライマリ（テラコッタオレンジ）

食欲と温かみを表現する深いオレンジ〜テラコッタ系。

| トーン | Hex | HSL | 用途 |
|--------|-----|-----|------|
| 50 | `#FFF5F0` | `20 100% 97%` | 薄い背景 |
| 100 | `#FFE8DB` | `20 100% 93%` | ホバー背景 |
| 200 | `#FFD0B8` | `20 100% 86%` | 軽いアクセント |
| 300 | `#FFB08A` | `20 100% 77%` | — |
| 400 | `#FF8A56` | `20 100% 67%` | — |
| 500 | `#E86A30` | `20 80% 55%` | **メインカラー** |
| 600 | `#C85420` | `20 73% 45%` | ホバー時 |
| 700 | `#A84218` | `20 75% 38%` | アクティブ時 |

## セマンティックカラー

| 意味 | ライトモード | ダークモード | 用途 |
|------|------------|------------|------|
| Success | `#3D8B4D` | `#5CB870` | 保存完了、成功通知 |
| Error | `#D64545` | `#F06565` | エラー、バリデーション失敗 |
| Warning | `#D4920A` | `#F0B840` | 注意喚起 |
| Info | `#3B7AC0` | `#60A0E0` | 情報通知、ヒント |

## ライトモード / ダークモード

### ライトモード

| 役割 | カラー | Hex |
|------|-------|-----|
| 背景 | neutral-50 | `#FAFAF8` |
| カード背景 | white | `#FFFFFF` |
| ボーダー | neutral-200 | `#E8E6E1` |
| メインテキスト | neutral-900 | `#1F1E1C` |
| セカンダリテキスト | neutral-500 | `#8A8680` |
| プライマリ | primary-500 | `#E86A30` |
| プライマリ foreground | white | `#FFFFFF` |

### ダークモード

| 役割 | カラー | Hex |
|------|-------|-----|
| 背景 | neutral-950 | `#121110` |
| カード背景 | neutral-800 | `#33312E` |
| ボーダー | neutral-700 | `#4D4A46` |
| メインテキスト | neutral-100 | `#F5F4F1` |
| セカンダリテキスト | neutral-400 | `#B0ACA3` |
| プライマリ | primary-400 | `#FF8A56` |
| プライマリ foreground | neutral-900 | `#1F1E1C` |
